  To adjust settings, edit main.py around line 83:

  WHISPER_MODEL = "tiny.en"        ← Change STT model
  LLM_MODEL = "llama3.2:3b"        ← Change LLM (llama3:8b, phi4-mini, etc.)
  RMS_THRESHOLD = 0.01             ← VAD sensitivity (lower = more sensitive)
  WAKEWORD_NAME = "alfred"         ← Change wake word
